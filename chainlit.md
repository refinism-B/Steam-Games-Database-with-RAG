## 🎮 Steam RAG 資料庫助理

歡迎使用！本系統專為 **Steam 玩家** 設計，結合 RAG 技術與大型語言模型，為您提供精準的遊戲資料查詢。

### 系統說明
* 目前僅支援**單一遊戲**問題查詢（未來預計更新多遊戲對比或查詢功能）。
* **free/Gemini 3 flash**模型使用免費的 API Key，很可能面臨額度耗盡無法使用的情況。
* **local/Gemma 3 12B**模型是連接地端 GPU 電腦運行，僅測試時使用。
* 想要完整體驗請選擇**price/Gemini 3 flash**模型，此為付費 API Key，請酌量使用。
